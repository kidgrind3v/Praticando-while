# 🧮 Calculadora em Java (Console)

Projeto simples de uma calculadora feita em Java para praticar lógica de programação, estruturas de repetição (`while`) e condicionais (`if/else`).



## 🚀 Funcionalidades

- ➕ Soma
- ✖️ Multiplicação
- ➗ Divisão (com validação de divisão por zero)
- ➖ Subtração
- 🔁 Menu interativo com loop (`while`)
- ⚠️ Tratamento de opção inválida



## 🧠 Conceitos praticados

- Estrutura de repetição (`while`)
- Estruturas condicionais (`if`, `else if`, `else`)
- Entrada de dados com `Scanner`
- Controle de fluxo
- Lógica de programação



## 💻 Como funciona

O programa exibe um menu no terminal:
1 - Somar
2 - Multiplicar
3 - Dividir
4 - Subtrair
0 - Sair


O usuário escolhe uma opção e informa dois números para realizar a operação.

## 🛠️ Tecnologias

- Java

## 📌 Objetivo

Projeto criado com o objetivo de praticar lógica de programação e consolidar o uso de loops e estruturas condicionais.
